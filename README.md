# Demo

予め発行した代用貨幣トークンを用いて、</br>
他ユーザ(送信者)によりタグの情報を変更された際に、</br>
タグの持ち主から他ユーザ(送信者)へトークンが支払われる。

## Openzeppelinライブラリを用いる
トークン発行・管理するため、ERC20.solをインポート

## Remix IDEを用いる
https://remix.ethereum.org/
